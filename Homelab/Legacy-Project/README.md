# Legacy Projects

This section contains projects that are no longer deployed within the current homelab environment.

Although retired, these projects played an important role in the evolution of the infrastructure and provided valuable learning opportunities.

## Projects

| Project                                         | Status   | Reason for Retirement                                |
| ----------------------------------------------- | -------- | ---------------------------------------------------- |
| [Nextcloud](nextcloud.md)                       | Retired  | Replaced by a dedicated NAS and Synology Drive       |
| [FileBrowser](filebrowser.md)                   | Retired  | No longer required after project completion          |
| [OpenVPN to WireGuard](openvpn-to-wireguard.md) | Migrated | Replaced by a simpler and more maintainable solution |

---

## Why Keep Legacy Projects?

Infrastructure evolves over time.

These projects demonstrate:

* Technology evaluation
* Architectural decision-making
* Service migrations
* Infrastructure simplification
* Lessons learned from previous deployments

Understanding why a service was retired is often as valuable as understanding why it was deployed.
